Cursor supports markdown out of the box and has partial built-in support for MDX. You can install the <a href="https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx" target="_blank">MDX extension</a> from the VS Code Marketplace for better syntax highlighting and LSP.

LSP stands for Language Server Protocol. It is a standard that code editors like Cursor communicate with a language server and provides smart features like **Autocomplete**, **Go to Definition**, etc.
